### APIs

- **Interna**

  - **Rotas**
    - Usuários
      - Autenticação

      - Edita usuário
      - Exibe perfil do usuário

      - Adiciona novos produtos
      - Lista todos os produtos
      - Lista somente dados de um produto selecionado
      - Edita dados de produtos existentes
      - Exclui produtos existentes
    - Pedidos
